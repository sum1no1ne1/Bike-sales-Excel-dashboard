Bike Sales Analysis Dashboard (Excel)
Overview

This project analyzes customer demographic and behavioral data to identify factors influencing bike purchases. The analysis was performed using Microsoft Excel, including data cleaning, feature engineering, and interactive dashboard creation.

Problem Statement

To understand which customer segments are more likely to purchase bikes based on attributes such as income, age, gender, commute distance, and lifestyle factors.

Dataset

The dataset contains customer-level information with the following features:

Demographics: Age, Gender, Marital Status, Region
Socioeconomic: Income, Education, Occupation
Lifestyle: Number of Children, Cars, Commute Distance
Target Variable: Bike Purchase (Yes/No)
Data Cleaning & Transformation
Standardized categorical variables:
Marital Status (M/S → Married/Single)
Gender (M/F → Male/Female)
Removed duplicate records to ensure data integrity
Created a derived column:
Age Brackets (Young, Middle Age, Old) for segmentation
Formatted income data for consistent numerical analysis
Analysis & Techniques
Used Pivot Tables to analyze relationships between:
Income and purchase behavior
Age groups and purchase distribution
Commute distance and buying patterns
Applied Excel functions such as IF, COUNTIF, and aggregation functions
Built interactive visualizations using charts and slicers
Dashboard Features
Interactive filters for:
Marital Status
Age Brackets
Number of Cars
Number of Children
Visual analysis of:
Average income by gender and purchase status
Purchase distribution across age groups
Commute distance vs purchase behavior
Key Insights
Customers with shorter commute distances showed higher likelihood of purchasing bikes
Income influenced purchase behavior differently across genders
Age-based segmentation helped identify variations in purchase patterns across customer groups
Tools Used
Microsoft Excel
Pivot Tables
Charts and Slicers
